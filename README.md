### How to start

#### Prerequisites
* [node](https://github.com/nodejs/node)

* [yarn](https://github.com/yarnpkg/yarn)

* [ionicframework](https://github.com/ionic-team/ionic) 

```
$ sudo npm install -g ionic cordova
```

#### Installing all the dependencies of project

```
$ yarn
```

#### Run an app

```
$ ionic serve -l
```
